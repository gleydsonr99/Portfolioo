Meus Projetos

Estácio Sports

Descrição: O projeto "Estácio Sports" é uma plataforma web dedicada à gestão e promoção de eventos esportivos. Desenvolvido para a Estácio, um renomado centro educacional, o site permite que os usuários explorem e se inscrevam em eventos esportivos, acompanhem resultados e interajam com a comunidade esportiva. O objetivo principal é fornecer uma experiência intuitiva e envolvente para todos os entusiastas do esporte, combinando informações sobre eventos com funcionalidades de interação social.

Funcionalidades Principais:

  • Calendário de Eventos: Visualize e filtre eventos esportivos por data, tipo e localização.
  • Inscrição em Eventos: Inscreva-se diretamente em eventos e receba confirmações e atualizações via e-mail.
  • Resultados e Estatísticas: Acompanhe os resultados dos eventos e visualize estatísticas detalhadas.
  • Perfil do Usuário: Crie e gerencie seu perfil, visualize seu histórico de inscrições e participe de discussões.
  • Notificações: Receba notificações sobre eventos futuros, mudanças e atualizações importantes.

Tecnologias Utilizadas: HTML, CSS, JavaScript, Node.js/Express, MongoDB/MySQL, APIs.

Objetivo: O principal objetivo do projeto "Estácio Sports" é criar uma plataforma centralizada onde os usuários possam facilmente descobrir, se inscrever e acompanhar eventos esportivos. A aplicação visa melhorar a experiência dos participantes, facilitando a gestão de eventos e promovendo a interação entre os membros da comunidade esportiva.

Buscar CEP

Descrição: O projeto "Buscar CEP" é uma aplicação web que permite aos usuários consultar endereços completos a partir de um código postal (CEP). A ferramenta foi desenvolvida para oferecer uma solução rápida e prática para a busca de endereços no Brasil, facilitando o acesso a informações de localização de forma eficiente e precisa.

Funcionalidades Principais:

  • Busca de Endereço: Consulte o endereço completo a partir de um código postal (CEP).
  • Interface Simples: Design minimalista e intuitivo para facilitar o uso.
  • Resposta Rápida: Fornecimento rápido e preciso das informações de endereço.

Tecnologias Utilizadas: HTML, CSS, JavaScript, API de Consulta de CEP.

Objetivo: O objetivo do projeto "Buscar CEP" é proporcionar uma maneira simples e eficaz para os usuários consultarem informações de endereços a partir de um código postal, com uma interface amigável e um desempenho rápido.

Calculadora One

Descrição: O projeto "Calculadora One" é uma aplicação web de calculadora com design moderno e funcional. Desenvolvida para proporcionar uma experiência de usuário intuitiva e eficiente, a calculadora permite realizar operações matemáticas básicas e avançadas com precisão e facilidade. O design limpo e a interface responsiva garantem que a calculadora funcione perfeitamente em qualquer dispositivo, seja no desktop ou em dispositivos móveis.

Funcionalidades Principais:

  • Operações Básicas: Realize operações matemáticas básicas, como adição, subtração, multiplicação e divisão.
  • Operações Avançadas: Inclui funcionalidades adicionais como cálculo de porcentagens e raízes quadradas.
  • Design Responsivo: Interface que se adapta a diferentes tamanhos de tela, garantindo uma boa experiência em qualquer dispositivo.
  • Histórico de Cálculos: Permite visualizar o histórico de operações realizadas durante a sessão.

Tecnologias Utilizadas: HTML, CSS, JavaScript.

Objetivo: O objetivo do projeto "Calculadora One" é fornecer uma ferramenta de cálculo simples e acessível, com um design que favoreça a usabilidade e a eficiência. A aplicação visa atender tanto a usuários que precisam de uma calculadora básica quanto àqueles que necessitam de funcionalidades matemáticas adicionais, tudo isso em uma interface amigável e responsiva.
